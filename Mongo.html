📦 El Formato BSON

En MongoDB 4, los datos se almacenan en formato binario. Esto permite:

  • Velocidad de escaneo.
  • Soporte de fechas reales.
  • _id único: Generado como ObjectId si no se especifica.

🚀 Operaciones CRUD en Consola

Insertar (Create)

Terminal mongo db.inventario.insertOne({
  item: "Tester",
  stock: 10
})

Actualizar (Update)

Uso de $set db.inventario.updateOne(
  { item: "Tester" },
  { $set: { stock: 15 } }
)
