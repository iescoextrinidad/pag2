Explorando el Mundo de Arduino

La plataforma líder para prototipado electrónico de código abierto

¿Qué es Arduino?

Arduino es una plataforma de creación de electrónica de código abierto (open-source), basada en hardware y software libre, flexible y fácil de utilizar para los creadores y desarrolladores. Esta plataforma permite crear objetos interactivos que pueden sensar el entorno y actuar sobre él.

Componentes Principales de una Placa

Microcontrolador: El "cerebro" que procesa las instrucciones (comúnmente ATmega328P).
Pines Digitales: Entradas/Salidas que solo entienden dos estados: encendido o apagado.
Pines Analógicos: Pueden leer un rango de valores (útil para sensores de temperatura o luz).
Puerto USB: Para cargar el código desde la computadora y alimentar la placa.

Ejemplo de Código: "Blink"

Este es el "Hola Mundo" de la electrónica, donde hacemos parpadear un LED conectado al pin 13.

// La función setup se ejecuta una vez al inicio
void setup() {
  pinMode(13, OUTPUT); // Configura el pin 13 como salida
}

// La función loop se repite infinitamente
void loop() {
  digitalWrite(13, HIGH); // Enciende el LED
  delay(1000);           // Espera un segundo
  digitalWrite(13, LOW);  // Apaga el LED
  delay(1000);           // Espera un segundo
}
